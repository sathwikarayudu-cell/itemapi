# This is a simple RESTful backend application built using Java Spring Boot.
It provides APIs to manage a collection of items (like products in an e-commerce system).

The application uses in-memory storage (ArrayList) and does not require a database.


Add a new item
Get item details by ID
Input validation (manual validation)
In-memory data storage
RESTful API design
Tech Stack:
Java 21
Spring Boot
Maven
REST API
In-memory storage using ArrayList
